# An Gun | Product Designer Portfolio

안건(An Gun) 디자이너의 공식 포트폴리오 사이트입니다. 사용자 경험(UX)과 인터페이스(UI) 설계 역량을 현대적인 감각으로 선보입니다.

## 🚀 주요 기능 및 특징

- **고급 애니메이션 (GSAP):** `ScrollTrigger`를 활용한 텍스트 리빌, 스크롤 인터랙션.
- **부드러운 스크롤 (Lenis):** `hilonishah.com` 스타일의 고급스러운 스크롤 경험.
- **커스텀 커서:** 사용자 인터랙션에 반응하는 자석(Magnetic) 효과 및 커서 변형.
- **반응형 디자인:** 모바일 및 태블릿 환경에 최적화된 레이아웃.
- **타이포그래피:** `Wanted Sans`, `Pretendard` 등 현대적인 폰트 적용.

## 🛠 사용된 기술

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Animation:** GSAP, ScrollTrigger, ScrollToPlugin
- **Scroll Engine:** Lenis
- **Typography:** SplitType
- **Design:** Modern UI/UX principles

## 📁 프로젝트 구조

- `index.html`: 메인 웹 사이트 구조
- `style.css`: 테마, 레이아웃 및 애니메이션 스타일
- `script.js`: 모든 인터랙션 및 애니메이션 로직
- `public/assets/`: 이미지 및 그래픽 자산
